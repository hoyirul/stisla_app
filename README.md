# stisla_app

## How to install
- make api server (laravel, lumen etc)
- git clone https://github.com/hoyirul/stisla_app.git
- cd stisla_app
- flutter pub upgrade
- edit your endpoint (lib/utils/api_endpoints.dart)
- flutter run
